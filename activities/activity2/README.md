# CST 391 – Activity 2: Angular Test Application

## Overview
This activity focuses on creating and exploring a basic Angular application using the Angular CLI. The goal is to understand the default project structure, run the application locally, and make simple modifications to the user interface.

---

## Project Location
The Angular test application for Activity 2 is located in: activities/activity2/testapp


---

## Tools and Technologies
- Angular CLI
- Node.js and npm
- TypeScript
- Visual Studio Code

---

## Steps Completed
- Installed and verified the Angular CLI
- Generated a new Angular test project using `ng new`
- Ran the application locally using `ng serve`
- Updated the application title
- Added a custom welcome message displaying the developer’s name
- Verified changes in the browser

---

## How to Run the Application
From the `testapp` directory, run:

```bash
ng serve -o

The application will launch in a browser at: http://localhost:4200
